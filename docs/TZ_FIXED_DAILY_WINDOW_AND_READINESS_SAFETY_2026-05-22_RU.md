# ТЗ фикс: 24ч train/test + безопасность readiness

Дата: 2026-05-22

## Что зафиксировано
1. Train только 1 полный UTC день: `train_start == train_end`.
2. Test только 1 полный UTC день: `test_end_day == test_day`.
3. Train-день обязан быть ровно предыдущим днем относительно test-дня.
4. Test-день обязан быть закрытым: не позже `UTC yesterday`.
5. Флаг `--stop-on-goal-pass` отключен политикой (игнорируется), перебор не обрезается.
6. Временный unlock readiness теперь безопаснее:
1. Пишется marker с исходным состоянием.
2. В `finally` readiness восстанавливается и marker удаляется.
3. Если прошлый run упал/был убит, следующий запуск автоматически поднимает marker и восстанавливает readiness.
