# Codex Project Memory

This folder is the managed memory for continuing `MLbotNav` work across new Codex chats.

Read these files at the start of a new chat:
1. `AGENTS.md`
2. `docs/codex/handoff.md`
3. `docs/codex/current_state.md`
4. `docs/codex/todo.md`
5. `docs/codex/known_errors.md`
6. `docs/codex/commands.md`

Use these files as project memory:
1. `handoff.md` - compact project handoff and next step.
2. `current_state.md` - what works, what is blocked, recent results.
3. `decisions.md` - important decisions and rejected paths.
4. `todo.md` - current task board.
5. `known_errors.md` - recurring errors and fixes.
6. `commands.md` - useful commands and safe run patterns.
7. `session_log.md` - compact session history.
8. `artifacts.md` - important files and why they matter.

Do not copy full chat logs into this folder. Keep only decisions, state, errors, commands, and artifact links.
