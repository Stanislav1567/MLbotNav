# Запуск STAS9

## Основной способ

Для обычной работы используйте:

```text
Рабочий стол -> 🤖 STAS9 Workspace
```

Ярлык открывает:

```text
C:\Users\007\Desktop\MLbotNav\MLbotNav_STAS9.code-workspace
```

в Visual Studio Code. После открытия отправьте команду в панели Codex:

```text
STAS9, подтверди режим SAFE и готовность. Ничего не изменяй.
```

STAS9 действует как роль `STAS9_SENTINEL` внутри текущей задачи Codex. Он
активируется только командой пользователя и не оставляет отдельный фоновый
процесс.

## Технический резерв

Файл `start_STAS9.bat` запускает консольный Codex CLI и сохранён только для
диагностики или ручного технического использования. Он не является основным
интерфейсом, не входит в автозагрузку и не должен работать постоянно.

## Безопасность

По умолчанию действует `SAFE + READ_ONLY`. Создание или изменение файлов,
эксперименты и другие расширенные действия требуют отдельного явного
разрешения пользователя. Обучение, Optuna, изменение моделей и реальная
торговля автоматически не запускаются.
