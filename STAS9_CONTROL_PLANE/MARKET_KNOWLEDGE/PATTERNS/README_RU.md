# Паттерны SOL

Каталог предназначен для описаний повторяемых конфигураций, обнаруженных в нескольких проверенных событиях SOL.

Описание паттерна должно включать:

- идентификаторы подтверждающих событий;
- обязательные и необязательные условия;
- контекст таймфрейма;
- условия опровержения;
- известные ложные срабатывания.

Одиночное событие не считается подтверждённым паттерном.
