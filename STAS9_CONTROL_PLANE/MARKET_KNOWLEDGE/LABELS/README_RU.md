# Метки событий SOL

Здесь хранятся версии правил, по которым проверенным событиям назначаются целевые метки.

Начальный словарь исследуемых исходов:

- хороший вход;
- плохой вход;
- ложный сигнал;
- продолжение движения;
- разворот.

Для каждой метки необходимо определить измеримый горизонт, порог движения, порядок учёта комиссий и правило разрешения неоднозначных случаев. До утверждения этих условий метка считается предварительной.
