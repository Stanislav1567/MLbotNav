# Паспорта наборов данных SOL

Каталог предназначен для паспортов версий ML-наборов данных.

Паспорт должен содержать:

- идентификатор и версию набора;
- перечень исходных событий;
- версии контрактов признаков и меток;
- временные границы;
- правила разбиения;
- проверки качества и утечек;
- контрольную сумму готового артефакта.

Шаблоны, непроверенные события и строки без происхождения данных исключаются. На первом этапе наборы только проектируются и проверяются, обучение не запускается.
