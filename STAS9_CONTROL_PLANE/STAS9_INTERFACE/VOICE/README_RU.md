# Голосовой ввод STAS9

## Рабочая схема

```text
Микрофон
-> системная диктовка Windows
-> русский текст в поле Codex
-> STAS9_SENTINEL
-> текстовый ответ
```

## Как говорить с STAS9

1. Открыть workspace через ярлык `🤖 STAS9 Assistant`.
2. Установить курсор в поле ввода Codex.
3. Нажать `Win+H`.
4. Убедиться, что выбран русский язык.
5. Произнести обычную фразу.
6. Проверить распознанный текст и отправить сообщение.

Пример:

```text
Проверь STAS5, найди сломанные указатели моделей.
```

Sentinel классифицирует её как `AUDIT_BROKEN_POINTER`, выбирает `STAS9_AUDITOR` и сохраняет текст команды в журнале.

## Конфиденциальность

- STAS9 не включает микрофон самостоятельно;
- STAS9 не сохраняет исходное аудио автоматически;
- в журнал попадает только текст после отправки пользователем;
- секреты и ключи нельзя диктовать в чат;
- доступ Windows к микрофону пользователь контролирует в системных настройках.
