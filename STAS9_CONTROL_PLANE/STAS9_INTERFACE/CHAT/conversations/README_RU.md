# Диалоги STAS9

Этот каталог предназначен для человекочитаемых сводок или экспортов диалогов, которые пользователь явно просит сохранить.

Основная история остаётся в интерфейсе Codex. Структурированный audit trail каждой команды записывается в:

```text
STAS9_CONTROL_PLANE/LOGS/conversations/
```

Рекомендуемое имя сводки:

```text
YYYY-MM-DD_HHMMSS_<conversation_id>_RU.md
```

Сводка не должна содержать секреты, токены, ключи API или необработанные большие журналы.
