# Ответы STAS9

Все ответы главного агента формируются на русском языке и соответствуют `RESPONSE_TEMPLATE_RU.md`.

Правила:

1. сначала сообщается текущий статус;
2. всегда указывается выбранный агент;
3. задача формулируется одной понятной фразой;
4. результат отделяется от плана и предположений;
5. перечисляются изменённые файлы либо явно пишется `Нет`;
6. при блокировке указывается требуемое подтверждение пользователя;
7. большие журналы и отчёты сохраняются в файл, а в чат возвращается краткая выжимка и путь.
