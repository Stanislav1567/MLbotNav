# Команды STAS9

Пользователь говорит или пишет обычными русскими фразами. Знание машинных идентификаторов не требуется.

`STAS9_SENTINEL` должен:

1. сохранить исходный текст команды;
2. определить намерение;
3. выбрать минимально необходимого агента;
4. объявить режим и ограничения;
5. выполнить разрешённые действия;
6. вернуть результат по шаблону из `../RESPONSES/RESPONSE_TEMPLATE_RU.md`.

Пример:

```text
Пользователь:
Проверь STAS5, найди сломанные указатели моделей.

Распознанная команда:
AUDIT_BROKEN_POINTER

Исполнитель:
STAS9_AUDITOR

Режим:
SAFE / READ_ONLY
```

Таблица маршрутизации находится в `COMMAND_ROUTING.yaml`. Она помогает классифицировать задачу, но не является разрешением на опасное действие.
