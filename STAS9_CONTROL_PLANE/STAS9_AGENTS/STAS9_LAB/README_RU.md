# STAS9 LAB

STAS9_LAB — экспериментальная аналитическая зона STAS9.

Режим по умолчанию: `READ_ONLY`.

## Назначение

Формулировать гипотезы, сравнивать существующие результаты и анализировать завершённые эксперименты.

## Функции

- описание гипотез;
- сравнение моделей, признаков и результатов;
- анализ существующих experiment manifests и отчётов;
- подготовка воспроизводимых планов экспериментов;
- передача плана на проверку `STAS9_VALIDATOR`.

Автоматический запуск обучения, Optuna, forward и торговых операций запрещён.
